## 📋 What is java-jacoco-coverage-backfill?

This tool helps you check how much of your Java code gets tested. It works with Java projects built using Maven or Gradle, which are popular tools for managing Java projects. The main purpose is to improve test coverage by filling in missing data using JaCoCo reports.

It offers features like branch coverage analysis, the ability to run custom workflows, and comparing snapshots to see what changed between test runs.

## 🔧 How to Use java-jacoco-coverage-backfill (Basic Guide)

This application mainly works with Java projects. You can use it to check and fill in test coverage information. Here is a simple overview of how it works once installed.

1. **Select Your Project**  
   Open the app and locate your Java project folder on your computer. This folder should contain files for Maven or Gradle projects, like `pom.xml` for Maven or `build.gradle` for Gradle.

2. **Start an Analysis**  
   The tool will look for JaCoCo coverage reports. JaCoCo creates reports that show which parts of your code got tested. If you don’t have such reports, the tool can help generate or estimate missing coverage data.

3. **View Coverage Details**  
   The app shows results about test coverage. It highlights which parts of your code are fully tested and which parts are not. It can also compare different runs to show changes over time.

4. **Use Scriptable Workflows (Optional)**  
   For advanced users, the tool supports custom workflows. These are sequences of tasks you can set to automate coverage analysis and reporting.

## 📥 Installing Java (If Needed)

Since this tool works with Java projects, having Java installed on your PC can help. This step may not be necessary if your project already includes required Java components.

To check if Java is on your PC:

- Open the **Command Prompt** (press Win + R, type `cmd`, press Enter).
- Type `java -version` and press Enter.
- If you see a version number, Java is installed. If you get an error, you should install it.

To install Java:

1. Visit the official Java download page via your browser: https://www.oracle.com/java/technologies/javase-downloads.html  
2. Download the Windows installer for the **Java SE Runtime Environment (JRE)** or **Java Development Kit (JDK)**.
3. Run the installer and follow the prompts.
4. After installation, repeat the version check to confirm.
